# 图片项目 Step3 模特图规划（幼龄版）

幼龄角色专属模特图拍摄规划，针对幼龄角色的特殊比例、皮肤质感和姿态需求优化。

## 核心特性

- **皮肤质感**：细腻自然、无成人毛孔纹理、无过度磨皮
- **体型比例**：头身比约1:5-1:6、四肢圆润、体型匀称
- **姿态风格**：自然活泼、避免成人化姿势、重心稳定
- **年龄安全**：模糊年龄描述、避免触发安全过滤器
- **专业姿势库**：27种专业电商姿势（基础展示、动态、静态、情境、特写）
- **专业背景库**：39种专业电商背景（纯色、渐变、室内、户外、特殊）
- **智能规划**：根据角色年龄、性别、风格智能匹配姿势和背景组合
