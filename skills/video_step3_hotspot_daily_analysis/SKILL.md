# 每日热点分析报告生成

## 用途

为 Step3 脚本生成提供每日预计算的热点分析报告，包含五个结构化段落。

## 输入参数

- `hotspotCount`: 热点数量
- `hotspotDetails`: 热点详细信息列表

## 输出格式

五段结构化文本：
1. 核心趋势
2. 穿搭切入点
3. 情绪氛围
4. 规避话题
5. 创意建议
