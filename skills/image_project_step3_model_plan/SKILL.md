# 图片项目 Step3 成人模特图规划

成人专属模特图拍摄规划，针对成人角色的皮肤质感、面部特征和姿态需求优化。

**核心特性**：

- **成人皮肤**：可见毛孔、自然纹理、真实质感
- **面部保真**：五官比例自然、对称性合理
- **姿势物理化**：重心明确、阻力感、重量感
- **光线真实**：45°侧光、柔和漫射、纹理保留
- **专业姿势库**：27种专业电商姿势（基础展示、动态、静态、情境、特写）
- **专业背景库**：39种专业电商背景（纯色、渐变、室内、户外、特殊）
- **智能规划**：根据角色年龄、性别、风格智能匹配姿势和背景组合
