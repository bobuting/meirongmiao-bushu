# 专业电商模特图背景库

## 纯色背景（简约专业）
1. **纯白背景**：纯白色背景，光线柔和均匀，适合展示服装本色，电商主图必备
2. **浅灰背景**：浅灰色背景，光线中性，适合中性色调服装，展现高级感
3. **米色背景**：米色背景，光线温暖，适合暖色调服装，展现温馨感
4. **深灰背景**：深灰色背景，光线低调，适合冷色调服装，展现沉稳感

## 风格适配建议
- **通勤风格**：办公室、城市街道、商业广场、会议室（60%）
- **休闲风格**：公园、咖啡馆、书店、家居客厅（50%）
- **运动风格**：健身房、瑜伽室、山地、海边沙滩（70%）
- **优雅风格**：艺术空间、餐厅、花园庭院、日落场景（60%）
- **潮流风格**：霓虹渐变、酒吧、屋顶、夜景城市（80%）
- **复古风格**：复古场景、书店、咖啡馆老店（50%）
