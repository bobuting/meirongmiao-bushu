# 产品展示/口播专用流程

> 电商视频核心场景的专用方法论，覆盖产品展示流程、口播与产品动作同步、产品载体策略。
> **前提**：仅当分镜类型为产品展示/口播/电商带货时才适用。手部动作和产品同步应从分镜数据中推导，不凭空创造原文没有的动作。

## 产品展示流程模板

产品展示类分镜按以下 6 阶段流程推进（根据分镜内容选择对应阶段）：

| 阶段 | 内容 | Prompt 要点 |
|------|------|------------|
| 1. Hook（吸引） | 产品/模特首次亮相 | 产品整体外观 + 吸引眼球的视觉特征 + 悬念构图（只展示部分/剪影/特写） |
| 2. 质感展示 | 近距离展示产品材质 | 微距级别细节 + 材质响应描述（光线在布料/皮革/金属上的表现）+ 手部触碰材质的动作 |
| 3. 使用演示 | 产品在场景中的使用状态 | 产品与人的交互动作 + 使用场景环境 + 产品功能点视觉化 |
| 4. 产品特写 | 关键设计细节放大 | 聚焦单一设计元素 + 精确描述细节特征 + 背景虚化突出产品 |
| 5. 效果展示 | 穿戴/使用后的整体效果 | 全身展示产品效果 + 整体搭配协调性 + 场景氛围配合 |
| 6. CTA（号召） | 引导行动的画面 | 面对镜头/指向产品 + 表情自信 + 构图引导视线到产品 |

## 口播与产品动作同步

口播视频中，台词内容与产品展示动作必须同步：

| 台词内容类型 | 配合的手部动作 | 产品展示状态 |
|-------------|--------------|------------|
| 介绍产品整体 | 双手摊开展示，掌心向上 | 产品完整可见，光线均匀 |
| 强调材质/手感 | 手指触摸面料表面，轻柔滑动 | 镜头推近到接触点，展示材质纹理 |
| 对比功能/设计 | 左右手各持对比物或手指指向切换 | 两个对比元素同时可见或快速切换 |
| 总结推荐 | 单手指向产品或手持产品展示 | 产品居中，背景干净，光线突出产品 |

## 产品载体策略

产品信息在视频中的呈现方式：

| 载体类型 | 适用场景 | 描述要点 |
|----------|---------|---------|
| 正面标签展示 | 产品有品牌LOGO/文字 | 正面朝镜头，文字清晰可读，光线避免反光遮挡 |
| 平整载体 | 产品需要平坦展示（服饰平铺） | 产品铺平无褶皱，从正上方或45度俯拍，均匀光线 |
| 穿戴展示 | 服饰类产品 | 模特穿着展示，强调合身度和搭配效果 |
| 后期叠加 | 产品信息需要后期添加文字 | 预留干净的视觉空间（空白区域/简洁背景），避免复杂纹理干扰后期合成 |

## 电商场景氛围模板

产品展示场景的典型氛围选择：

| 产品风格 | 推荐氛围 | 光线方案 |
|----------|---------|---------|
| 高端奢侈品 | 奢华质感，暗调背景，金色点缀 | 戏剧光配方，强对比 |
| 日常休闲 | 明亮轻松，自然场景，生活化 | 户外自然光配方，柔和 |
| 专业功能 | 干净简洁，科技感，突出产品 | 产品展示光配方，均匀 |
| 温暖居家 | 暖色调，舒适场景，生活气息 | 窗口光配方，温暖 |
