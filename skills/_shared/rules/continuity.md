## 故事线完整连贯（不可违反）

**每个镜头都不是孤立的，它们共同讲述一个完整的故事。** 你必须确保跨镜头的故事连贯性。

### 故事线三要素

| 要素 | 含义 | 检查方法 |
|------|------|----------|
| **动作连贯** | 上一镜头结尾的动作状态 = 下一镜头开头的动作状态 | 镜头 N 结尾"低头看书" → 镜头 N+1 开头必须是"低头看书"或从该姿态开始的自然延续 |
| **情绪连贯** | 情绪变化有因果逻辑，不能跳跃 | "平静"不能直接跳到"狂喜"，中间必须有过渡 |
| **空间连贯** | 人物位置、朝向、与环境的相对关系在相邻镜头间保持一致或合理变化 | 镜头 N 人物在窗边左侧 → 镜头 N+1 不能突然出现在房间右侧 |

### 跨镜头衔接检查清单

生成每个镜头的视频 prompt 时，必须检查与前后镜头的衔接：

```
镜头 N 检查项：
□ 与镜头 N-1 的衔接：
  - 上一镜头结尾姿态/表情 = 本镜头开头姿态/表情？
  - 上一镜头结尾的空间位置 = 本镜头开头的空间位置？
  - 情绪是否有合理的延续或过渡？

□ 与镜头 N+1 的衔接：
  - 本镜头结尾姿态/表情 = 下一镜头开头姿态/表情？
  - 本镜头的动作是否为下一镜头留出自然衔接点？
```

### 常见连贯性错误

| 错误类型 | 错误示例 | 正确做法 |
|----------|----------|----------|
| **姿态跳变** | 镜头1结尾"坐着"，镜头2开头"站着" | 中间必须有起立动作，或明确标注时间跳跃 |
| **服饰跳变** | 镜头1 clothing.ref="搭配1"，镜头2突然变为"搭配2" | 除非分镜明确要求换装，否则同一场景内 clothing.ref 必须一致，clothing_features 必须逐字一致，keyframe prompt 中必须包含 clothing_features 形成文本+参考图双重锚点 |
| **情绪跳变** | 镜头1"悲伤"，镜头2突然"开心" | 必须有情绪过渡过程（悲伤→平静→释然→开心） |
| **空间跳变** | 镜头1在"窗边"，镜头2突然在"门口" | 除非有转场或移动动作，否则空间位置保持一致 |
| **光线跳变** | 镜头1"左侧逆光"，镜头2"正面平光" | 光线方向在相邻镜头间应保持一致或合理渐变 |
| **景别跳变** | 镜头1"远景"，镜头2突然"特写" | 相邻镜头景别变化不超过两级，避免突兀的视觉跳跃 |

### 景别变化规则（重要）

**景别变化必须平滑，禁止突然的大跨度切换。**

#### 景别等级定义

| 等级 | 景别 | 画面范围 |
|------|------|----------|
| 1 | 远景/大全景 | 全身+环境，人物占画面1/4以下 |
| 2 | 全景 | 全身，人物占画面1/2左右 |
| 3 | 中景 | 膝盖/腰部以上 |
| 4 | 近景 | 胸部以上 |
| 5 | 特写 | 面部或局部 |

#### 景别变化规则

| 规则 | 说明 | 示例 |
|------|------|------|
| **逐级变化** | 相邻镜头景别变化不超过2级 | 远景→全景→中景（✓）；远景→特写（✗ 跨度太大） |
| **避免跳切** | 禁止从远景直接跳到特写 | 会造成视觉突兀，观众迷失空间感 |
| **合理过渡** | 大景别变化需要中间镜头过渡 | 远景→中景→近景→特写 |

#### 正确的景别序列示例

```
✓ 正确：远景 → 全景 → 中景 → 近景（逐级推进）
✓ 正确：特写 → 近景 → 中景（逐级拉远）
✓ 正确：中景 → 近景 → 中景（小幅变化）
✗ 错误：远景 → 特写（跨度过大）
✗ 错误：全景 → 特写 → 全景（反复跳切）
```

#### 在视频 prompt 中的处理

如果输入分镜的景别变化过大，应在 `notes` 中标注风险：
```
"notes": "⚠️ 景别变化较大（远景→特写），建议中间增加中景过渡镜头"
```

### 故事线完整性验证

在生成所有镜头后，必须进行故事线完整性验证：

1. **动作链检查**：将所有镜头的动作串联，是否形成完整、自然的动作序列？
2. **情绪弧检查**：情绪变化是否符合"起因→发展→高潮→结局"的叙事逻辑？
3. **空间链检查**：人物在空间中的移动是否有合理的路径？
4. **时间链检查**：时间线是否清晰？是否有不合理的时间跳跃？
