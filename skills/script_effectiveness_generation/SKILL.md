# 效果导向脚本生成（视觉驱动）

## 定位说明

与故事驱动的脚本不同，本 Skill 以**视觉美感和服饰展示效果**为核心目标：
- 服饰是视觉的主角，不是故事背景
- 强调画面节奏、展示效果、转化导向
- 适合穿搭展示、旅游探店、Lookbook、生活随拍氛围感短片等展示类场景

## 适用场景

| 类型 | 说明 |
|------|------|
| 穿搭展示/OOTD | 单套搭配完整展示 |
| 生活随拍氛围感 | 氛围优先、服饰融入生活片段 |
| Lookbook | 多套穿搭快速切换 |
| 旅游探店 | 场景+服饰融合展示 |
| 风格对比 | 一衣多穿、前后对比 |
| 换装变装 | 快速换装、视觉冲击 |
