# 达人评估

## 核心定位

识别符合「场景种草」理念的抖音达人，这类达人创作的视频应该让观众在不知不觉中被种草，而不是直接的硬广推销。

## 评估标准

### 接受的内容类型

1. **生活美学短片 (aesthetic)**
   - 氛围感强，追求视觉美学
   - 没有说话，纯画面叙事
   - 服装作为生活场景中的视觉元素存在
   - 典型场景：咖啡店看书、公园散步、工作室创作

2. **时尚影像 (fashion_film)**
   - 电影感强，类似时尚杂志大片
   - 精致的光影构图
   - Lookbook 风格的穿搭展示
   - 典型场景：城市街头、艺术空间、建筑背景

3. **场景叙事 (scene)**
   - 有完整的故事线或场景转换
   - 服装随场景变化而自然切换
   - 情绪和故事感强
   - 典型场景：从家里到办公室、从白天到夜晚

### 拒绝的内容类型

1. **硬广类**
   - 直接推销产品
   - 夸张的"买它"、"必买"话术
   - 带有商品链接口播引导
   - 优惠券、限时促销等营销话术

2. **产品评测类**
   - 真人出镜讲解产品
   - "这件衣服怎么样"、"穿起来舒服吗"等询问式内容
   - 面对镜头的服装展示和讲解

3. **直播带货类**
   - 直播片段剪辑
   - 专门的带货账号
   - 明显的电商转化导向

## 评估维度

- **创作者昵称**：是否体现美学、时尚、生活等关键词
- **近期视频描述**：内容风格是否一致
- **置信度评分**：0-1，基于风格一致性
