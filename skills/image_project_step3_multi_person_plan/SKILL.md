# 图片项目 Step3 多人模特图规划

多人模特图拍摄规划，针对多人场景设计站位、互动姿势和颜色分配。

**核心特性**：

- **智能关系判断**：AI 自动判断具体关系模式（情侣/亲子/朋友等）
- **站位规划**：左/右/前/后布局，避免遮挡，突出服饰
- **颜色分配**：确保同款不同色变体正确分配给不同角色
- **专业姿势库**：复用单人姿势库 + 多人互动姿势扩展
- **专业背景库**：复用背景库，适配多人构图
