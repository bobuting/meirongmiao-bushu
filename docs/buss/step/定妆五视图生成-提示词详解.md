### 一、角色定义（ROLE）

```markdown
You are creating a professional character reference sheet (turnaround) 
for a character design project. Generate the turnaround based on the 
character preset direction, outfit information, and reference images 
provided by the user.
```

**定位**：专业的角色设计参考图生成器

---

### 二、任务定义（TASK）

```
+-----------+-----------+-----------+-----------+-----------+
| Close-up  |   Front   |   Left    |   Right   |   Back    |
|  (face)   |  (full)   |  (side)   |  (side)   |  (full)   |
+-----------+-----------+-----------+-----------+-----------+
```

**五视图布局**：
1. 特写（面部细节）
2. 正面（全身）
3. 左侧面（全身）
4. 右侧面（全身）
5. 背面（全身）

---

### 三、东方美学核心原则（EASTERN AESTHETICS CORE PRINCIPLES）

这是提示词的**核心差异化部分**，专门针对东方/亚洲审美：

#### 面部特征

| 特征 | ✅ 推荐 | ❌ 禁止 |
|------|--------|--------|
| **脸型** | 椭圆/圆/心形脸，柔和下颌线 | 方形下巴，V型下巴，婴儿肥 |
| **眼睛** | 单眼皮/内双，细长眼，卧蚕 | 欧式双眼皮，挑眉 |
| **鼻子** | 小直鼻，微驼峰，柔和鼻尖 | 夸张的混血鼻 |
| **嘴唇** | 薄唇，M型唇，自然血色 | 厚唇，光泽唇釉 |
| **肤色** | 暖白/中性一白，哑光微红 | 冷白，古铜色，油光 |

#### 整体气质

| 属性 | ✅ 推荐 | ❌ 禁止 |
|------|--------|--------|
| **皮肤质感** | 细腻但可见毛孔 | 完美磨皮 |
| **发型** | 黑长直/自然微卷/低马尾 | 浅色染发，齐刘海 |
| **气质** | 温婉，安静，书卷气 | 攻击性，性感 |
| **身材** | 纤细，标准比例 | 肥胖 |
| **体态** | 端正挺拔 | 驼背，含胸 |

### 四、生成规则（GENERATION RULES）

#### 1. 角色参考规则

```markdown
- The character's physical appearance (face, body, expression) is determined 
  **solely by the character preset direction text** provided in the user prompt.
- Do NOT copy the face or body from the outfit reference image.
- Apply Eastern aesthetics principles to all physical features.
```

**关键点**：
- 角色外貌**完全由文本描述决定**
- **禁止**从服饰参考图复制面部/身体
- 必须应用东方美学原则

#### 2. 服饰参考图规则

```markdown
- The user-provided outfit reference image is used **only as a reference 
  for the outfit/garments**.
- **Strictly ignore** any person, face, body, environment, background, 
  or any non-clothing elements.
- Maintain **high consistency** of the outfit across all 5 views.
```

**关键点**：
- 服饰参考图**仅用于服饰参考**
- **严格忽略**：人物、面部、身体、环境、背景
- **高一致性**：5 个视角的服饰细节必须完全一致（面料、颜色、图案、缝线、纽扣、拉链、配饰）

#### 3. 服饰文本一致性

```markdown
- Combine the outfit reference image with the user-provided outfit description text.
- The outfit description text includes complementary matching items — 
  all items (primary + complementary) must appear consistently across all 5 views.
- Do NOT modify, simplify, omit, or hallucinate any clothing features.
```

**关键点**：
- 结合服饰图 + 服饰文本描述
- 所有搭配单品必须在 5 个视角中一致出现
- **禁止修改、简化、遗漏、臆造**服饰特征

---

### 五、真实感指南（REALISM GUIDELINES）

这是**避免 AI 味**的核心约束：

#### 皮肤质感（必须包含）

```markdown
- **Natural skin texture**: visible pores, micro-details, realistic surface
- **Subtle imperfections**: slight freckles, fine lines around eyes, 
  minor skin tone variation
- **Micro-details**: peach fuzz on jawline, subtle oily highlights 
  on forehead/T-zone
- **Realistic lighting**: soft natural light that reveals skin texture, 
  NOT flat or over-smoothed
```

#### 表情与生命力

```markdown
- Subtle, natural expression — not a blank stare or forced smile
- Eyes should have depth and life, not glassy or dead-looking
- Slight asymmetry in facial features is natural and should be preserved
```

#### 整体质量

```markdown
- Commercial fashion photography aesthetic
- Real human, not a doll or CGI character
- Skin should look touchable, not rendered
```

## 🎯 核心设计理念

### 1. 东方美学优先

提示词**专门针对东方审美设计**，从脸型、眼睛、鼻子、嘴唇、肤色到气质都有明确约束，**避免西化、混血化**的 AI 生成结果。

### 2. 真实感至上

通过详细的皮肤质感、毛孔、细纹、不对称性约束，**避免典型的 AI 面部问题**（过度光滑、塑料感、空洞表情）。

### 3. 多视角一致性

5 个视角的服饰细节必须**完全一致**，包括面料、颜色、图案、缝线、纽扣、拉链、配饰等所有细节。

### 4. 角色与服饰分离

- 角色外貌**完全由文本描述决定**
- 服饰参考图**仅用于服饰参考**，不影响角色面部和身体

## ⚠️ 潜在问题分析

### 1. Logo/商标保持问题

**现状**：
- 负面提示词中包含 `watermark, text, logo`，**但这是禁止生成 logo**
- 提示词中没有明确要求**保持服饰上的品牌 logo**

**影响**：
- 如果参考图中的服饰有品牌 logo，生成结果可能会丢失或变形

**建议**：
- 参考服饰平铺图的优化方案，新增 Logo 保持约束

---

### 2. 多视角一致性挑战

**现状**：
- 提示词要求"Maintain **high consistency** of the outfit across all 5 views"
- 但**没有具体的技术指导**如何实现一致性

**影响**：
- 5 个视角的服饰细节可能出现细微差异（如纽扣数量、图案位置）

**建议**：
- 增加更具体的细节一致性约束，如：
  ```
  - 纽扣数量、位置、大小在 5 个视角必须完全一致
  - 图案/印花的形状、位置、比例在 5 个视角必须完全一致
  - 配饰（项链、手链、耳环）在 5 个视角必须同时出现/消失
  ```

---

### 3. 真实感约束可能不足

**现状**：
- 提示词要求"Real human, not a doll or CGI character"
- 但**缺少具体的视觉参考**或示例

**影响**：
- AI 模型可能仍然倾向于生成"完美化"的面部

**建议**：
- 增加更具体的技术约束，如：
  ```
  - 皮肤毛孔可见度：中等（不要太明显，也不要完全消失）
  - 面部不对称性：轻微（左右眼大小差异 < 5%，嘴角高度差异 < 3%）
  - 肤色不均匀度：轻微（额头略油，脸颊略干，T 区微亮）
  ```

---

### 4. 角色与服饰参考图冲突处理不明确

**现状**：
- 提示词要求"Do NOT copy the face or body from the outfit reference image"
- 但**没有明确如何处理冲突**（如角色描述与服饰参考图中模特的面部特征不一致）

**影响**：
- AI 模型可能在两者之间"混淆"，导致角色特征不稳定

**建议**：
- 增加优先级规则，如：
  ```
  **Priority Order** (highest to lowest):
  1. Character preset text description (highest priority)
  2. Character reference image
  3. Outfit text description
  4. Outfit reference image (lowest priority, ONLY for clothing details)
  ```
