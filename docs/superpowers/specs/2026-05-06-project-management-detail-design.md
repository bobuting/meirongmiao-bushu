# ProjectManagement 详情弹窗优化设计规格

> 创建时间：2026-05-06
> 目标用户：运营人员、业务管理者
> 信息深度：标准版（关键数据，各 Step 核心产出）

---

## 1. 设计目标

将现有的 `ProjectDetailModal` 从"基本信息+任务+资源"三标签页，升级为完整的项目流程可视化界面，支持查看各 Step 的关键产出数据。

**核心价值：**
- 一站式查看项目从 Step1 到 Step6 的所有关键数据
- 运营人员快速定位项目问题和产出质量
- 业务管理者了解项目整体进展

---

## 2. 弹窗结构

### 2.1 布局

```
┌─────────────────────────────────────────────────────────────┐
│ 项目详情                                              [X]   │
├─────────────────────────────────────────────────────────────┤
│ 基本信息（顶部固定，约 150px）                               │
│ - 项目封面 | 服饰主图                                        │
│ - ID、标题、类型、状态、当前Step、公司、用户、时间            │
│ - 操作按钮（解锁脚本/角色/服装搭配）                          │
├─────────────────────────────────────────────────────────────┤
│ 标签页导航                                                   │
│ [Step1] [Step2] [Step3] [Step4] [Step5] [Step6] [任务] [资源]│
├─────────────────────────────────────────────────────────────┤
│ 标签页内容区域（可滚动）                                      │
│ - 根据项目类型（视频/图片）展示不同内容                       │
│ - 每个标签页展示该 Step 的关键产出数据                        │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 尺寸

- 弹窗宽度：`max-w-5xl`（约 1024px）
- 弹窗高度：`max-h-[90vh]`
- 基本信息区：约 150px（固定）
- 标签页导航：约 50px（固定）
- 内容区域：flex-1 overflow-y-auto（可滚动）

## 4. 标签页内容

### 4.1 标签页顺序

| 标签页 | 适用项目 | 主要内容 |
|--------|----------|----------|
| Step1 | 视频/图片 | 服装上传/搭配推荐 |
| Step2 | 视频/图片 | 角色定妆 |
| Step3 | 视频/图片 | 脚本+分镜 / 模特图生成 |
| Step4 | 视频/图片 | 视频生成/合成 / 详情页生成 |
| Step5 | 视频/图片 | 发布相关 |
| Step6 | 仅视频 | 分镜（图片+视频） + 裂变视频 |
| 任务列表 | 全部 | 后台任务状态（后续优化） |
| 资源消耗 | 全部 | LLM/图片/视频调用统计 |

## 5. 技术实现要点

### 5.1 数据获取

**新增 API 接口（扩展现有 `getAdminProjectDetail`）：**

```typescript
interface ProjectDetailResponse {
  basicInfo: {...};
  characters: {...};
  tasks: {...};
  resourceConsumption: {...};

  // 新增：各 Step 数据
  step1Data: {
    garments: Garment[];
    matches: GarmentMatch[];
  };
  step2Data: {
    characters: Character[];
    selectedCharacterId: string;
  };
  step3Data: {
    script: Script | null;
    storyboards: Storyboard[];
    // 图片项目
    modelPhotos: ModelPhoto[];
  };
  step4Data: {
    clipVideos: ClipVideo[];
    finalVideo: FinalVideo | null;
    // 图片项目
    detailPages: DetailPage[];
  };
  step5Data: {
    publishRecords: PublishRecord[];
  };
  step6Data: {
    fissionStoryboards: FissionStoryboard[];  // 分镜（图片+视频）
    fissionVideos: FissionVideo[];            // 裂变视频
  };
}
```

### 5.2 缩略图处理

- 图片缩略图：`getOssThumbnailUrl(url, width, height)`
- 视频快照：`getOssVideoSnapshotUrl(videoUrl, timeSeconds, width)`

### 5.3 预览功能（所有标签页统一）

**所有标签页的视频和图片预览统一使用以下组件：**

- 图片预览：`ImageLightbox` 组件
- 视频预览：`VideoPreviewModal` 组件

**适用范围：**
- Step1：服饰图片
- Step2：角色缩略图、定妆效果图
- Step3：分镜图片、模特图（图片项目）
- Step4：分镜视频、最终合成视频、详情页图片（图片项目）
- Step5：发布封面
- Step6：分镜图片、分镜视频、裂变视频

### 5.4 缩略图处理（所有标签页统一）

**所有标签页的缩略图统一使用 OSS 优化：**

- 图片缩略图：`getOssThumbnailUrl(url, width, height)`
- 视频快照：`getOssVideoSnapshotUrl(videoUrl, timeSeconds, width)`

**适用范围：**
- 所有图片卡片缩略图
- 所有视频卡片封面

### 5.5 样式复用

- 网格布局：参考 `Step6FissionScreen` 的视频网格样式
- 卡片样式：圆角边框 + 状态标签 + 操作按钮
