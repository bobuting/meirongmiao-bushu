# 视频项目用户操作指南设计规格

## 概述

为内容喵（AI 电商短视频生成平台）编写用户操作文档，帮助电商商家理解视频项目完整流程，提供最佳实践建议，强调前期准备的重要性（项目类型选择后不可撤回）。

## 需求确认

| 项目 | 决策 |
|------|------|
| 目标用户 | 电商商家（不懂技术，需要快速上手） |
| 核心目的 | 最佳实践建议，强调前期准备重要性 |
| 文档形式 | 步骤清单 + 关键节点案例说明 |
| 项目范围 | 先做视频项目（后续图片项目复用模板） |
| 交付形式 | 独立 HTML 网页（无需登录即可查看） |
| 存放位置 | `public/user-guide/` |
| 方案数量 | 两份文档对比测试效果 |

## 交付物

### 文件结构

```
public/user-guide/
├── video-project-guide.html    # 方案 A：流程指南版
├── video-project-faq.html      # 方案 C：常见问题版
└── assets/                     # 资源文件（可选）
    └── images/                 # 后续可放置真实截图
```

### 访问方式

- 流程指南版：`http://your-domain/user-guide/video-project-guide.html`
- 常见问题版：`http://your-domain/user-guide/video-project-faq.html`

- 问题卡片样式：左侧编号 + 右侧答案

- 问题卡片 hover 效果：轻微上浮阴影

## 视觉设计规范

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 | #e68c19 | 橙色，与系统品牌一致 |
| 辅助色 | #fdfbf7 | 浅米色背景 |
| 文字色 | #333333 | 深灰主文字 |
| 强调色 | #22c55e | 绿色，成功/正确提示 |
| 警告色 | #ef4444 | 红色，重要提醒 |

### 排版规范

| 元素 | 规格 |
|------|------|
| 字体 | 系统默认中文字体（苹方/微软雅黑） |
| 主标题 | 24px 加粗 |
| 正文 | 16px 行高 1.8 |
| 清单项 | 18px |

### 组件样式

| 组件 | 样式 |
|------|------|
| 清单框 | 橙色边框圆角卡片，左侧勾选图标 |

| 案例块 | 浅橙背景，带故事图标 |
| 表格 | 橙色表头白字，交替灰白行，细灰边框 |

### 响应式断点

| 设备 | 宽度 | 特性 |

| 手机 | 375px | 折叠目录，单列布局 |
| 平板 | 768px | 侧边目录，双列卡片 |
| 电脑 | 1280px | 固定目录栏，宽松布局 |

## 技术实现方案

### 技术选型

| 特性 | 说明 |
|------|------|
| 单文件独立运行 | HTML + CSS 内嵌，无需外部依赖 |
| 无 JavaScript | 纯 CSS 交互效果，加载更快 |
| 响应式设计 | 媒体查询适配三种设备 |
| SEO 优化 | 语义化标签、meta 描述、标题层级 |
| 打印友好 | @media print 样式支持打印 |

### 后续维护方式

1. 更新内容：直接修改 HTML 文本
2. 添加截图：放入 `assets/images/`，修改 `<img>` src
3. 调整样式：修改 `<style>` 块 CSS
4. 新增问题：复制问题卡片模板，修改内容

## 成功标准

| 指标 | 目标 |
|------|------|
| 用户反馈 | 新用户阅读后能独立完成第一个视频项目 |
| 对比效果 | 两个方案收集用户偏好数据，确定最终版本 |
| 维护效率 | 后续内容更新无需技术团队介入 |
| 加载速度 | HTML 文件小于 100KB，加载时间小于 1 秒 |

## 风险与缓解

| 风险 | 缓解措施 |
|------|---------|
| 业务细节不准确 | 用户确认后填充，标注待补充区域 |
| 用户不看文档 | 首次创建项目时弹窗提示阅读 |
| 内容更新滞后 | 建立内容更新流程，与功能迭代同步 |
