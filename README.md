# 内容喵 AI

**AI 电商短视频/图片生成平台**

帮助电商从业者快速生成高质量的短视频、模特图和电商详情页。

---

## 一、核心功能

### 1.1 视频项目制作

完整短视频制作流程，从服装搭配到成片输出：

```
上传服装 → 智能搭配推荐 → 角色生成 → 脚本创作 → 分镜制作 → 视频合成 → 裂变扩展
```

**用户操作流程**：

**Step1 服饰搭配**：
- 上传服饰图片（上衣、下装、配饰等）
- 系统自动识别服饰属性（类别、款式、颜色）
- AI 推荐搭配方案（多套搭配可选）
- 选择角色预设方向（年龄、风格、场景）

**Step2 角色定妆**：
- 选择角色年龄和类型（儿童/成人）
- 系统批量生成角色五视图（正面/侧面/背面等）
- 选择最佳角色版本
- 确认角色外观（确保后续所有镜头一致）

**Step3 脚本+分镜**：
- 选择脚本策略（10+ 种策略可选）
- AI 自动生成完整脚本（故事线、台词、镜头描述）
- 系统生成分镜参考图（每个镜头的画面示意）
- 选择最佳分镜方案

**Step4 视频生成**：
- 批量生成分镜视频片段
- 选择每个镜头的最佳版本
- 添加背景音乐
- 合成完整视频

**Step5 发布准备**：
- 选择封面图
- 设置标题和标签
- 准备发布到电商平台

**Step6 裂变扩展**：
- AI 基于原脚本生成新故事线
- 批量生成裂变视频（6 或 12 个变体）
- 一次创作产出多个内容

---

### 1.2 图片项目制作

电商模特图和详情页制作流程：

```
上传服装 → 智能搭配 → 角色生成 → 模特图批量生成 → 电商详情页制作
```

**用户操作流程**：

**Step1 服饰搭配**：
- 上传服饰图片
- AI 提取产品卖点（材质、风格、适用场景）
- 推荐搭配方案

**Step2 角色定妆**：
- 生成角色五视图
- 选择最佳角色

**Step3 模特图生成**：
- 批量生成模特图（多姿势、多背景）
- 选择最佳模特图

**Step4 电商详情页生成**：
- 配置品牌 Logo（位置、大小、透明度）
- 生成各区块图片（产品介绍、卖点展示、场景展示）
- 一键生成拼接长图

---

### 1.3 换装项目制作

快速更换视频中角色服装：

```
上传源视频 → 确认视频内容 → 选择新服装 → 一键换装生成
```

**用户操作流程**：

**Step1 上传源视频**：
- 上传已有视频素材

**Step2 确认视频**：
- 系统分析视频内容
- 提取角色和场景信息

**Step3 选择角色**：
- 选择需要换装的角色

**Step4 一键换装**：
- 选择新服装
- 系统自动替换视频中的服装
- 输出新视频

---

### 1.4 逆向脚本项目

参考优秀视频，快速借鉴创作：

```
上传参考视频 → AI 反推脚本 → 生成分镜 → 创作新视频
```

**用户操作流程**：

**上传视频**：
- 上传优秀视频案例

**AI 分析**：
- 系统自动分析视频结构
- 提取脚本框架（镜头类型、时长分布、画面描述）
- 生成分镜逻辑

**创作新视频**：
- 借鉴脚本结构
- 替换为自己的产品和角色
- 生成新视频

## 二、脚本生成策略（10+ 种）

系统提供多种脚本策略，适应不同营销场景：

### 2.1 Library 策略（库存改写）

**功能**：改写已有优秀脚本，快速生成新内容

**用户操作**：
1. 系统从脚本库选择优秀脚本
2. 替换产品描述和角色信息
3. 保留脚本结构（镜头、时长、节奏）
4. 输出新脚本

**适用场景**：
- 快速生成脚本，节省创作时间
- 复用优秀脚本结构，提升质量

---

### 2.2 Video 策略（热榜借鉴）

**功能**：借鉴热门视频结构，提升流量潜力

**用户操作**：
1. 系统实时抓取抖音热榜视频
2. AI 分析视频结构（镜头、画面、节奏）
3. 替换为自己的产品和角色
4. 生成借鉴热榜结构的新脚本

**适用场景**：
- 借鉴热门视频流量密码
- 学习优秀视频节奏和结构

---

### 2.3 Realtime 策略（实时热点）

**功能**：结合实时热点事件，生成时效性脚本

**用户操作**：
1. 系统每小时抓取热点榜数据
2. AI 分析热点适配度（是否适合植入产品）
3. 选择高适配热点
4. 生成热点脚本（热点故事线 + 产品植入）

**适用场景**：
- 借势热点事件，提升传播力
- 时效性营销，快速响应热点

---

### 2.4 Effectiveness 策略（效果导向）

**功能**：突出产品效果对比，强说服力脚本

**脚本结构**：
```
开场（痛点引入） → 产品展示（效果对比） → 结尾（信任背书）
```

**用户操作**：
1. 系统识别产品痛点（如护肤品：皱纹、暗沉）
2. 设计效果对比场景（Before vs After）
3. 添加信任背书（用户评价、专家推荐）
4. 生成效果对比脚本

**适用场景**：
- 功能性产品（护肤品、保健品）
- 问题解决型产品（祛皱霜、美白精华）

---

### 2.5 Custom 策略（场景化种草）

**功能**：生活场景植入产品，自然真实种草

**场景类型**：
- 家庭场景（客厅、卧室、厨房）
- 职场场景（办公室、会议室）
- 户外场景（公园、咖啡馆、商场）
- 社交场景（聚会、约会、派对）

**用户操作**：
1. 系统根据产品类别自动匹配场景
   - 家居用品 → 家庭场景
   - 职场服装 → 职场场景
   - 休闲服装 → 户外场景
2. 设计场景故事（产品自然出现）
3. 生成场景化脚本

**适用场景**：
- 生活化产品（家居用品、日常服装）
- 自然植入，避免硬广感

---

### 2.6 Fashion 策略（时尚大片）

**功能**：T 台走秀风格，高端品牌感脚本

**时尚风格**：
- 都市优雅（职场、成熟）
- 青春活力（街头、潮流）
- 复古经典（优雅、知性）
- 前卫先锋（个性、张扬）

**用户操作**：
1. 系统根据服饰风格自动匹配时尚风格
2. 设计走秀镜头（T 台、pose、镜头运动）
3. 生成时尚大片脚本

**年龄适配**：
- 婴幼儿/幼儿：不可用（无法完成专业 pose）
- 儿童：可用（简化 pose）
- 青少年/成人：全可用

---

### 2.7 Emotion Archetype 策略（情感原型）

**功能**：基于情感原型理论，生成情感共鸣脚本

**情感原型**：
- 励志原型（奋斗成长、逆袭成功）
- 温情原型（亲情、友情、爱情）
- 怀旧原型（童年、青春、故乡）
- 梦想原型（追梦、理想实现）

**用户操作**：
1. 系统根据产品类别选择情感原型
   - 礼品 → 温情原型
   - 纪念品 → 怀旧原型
   - 运动装备 → 励志原型
2. 设计情感故事线
3. 生成情感共鸣脚本

**适用场景**：
- 情感共鸣产品（礼品、纪念品）
- 建立品牌情感连接

---

### 2.8 Aesthetic 策略（生活美学）

**功能**：展现产品美感，提升品牌调性

**美学风格**：
- 极简（简洁、清爽）
- 优雅（精致、高雅）
- 自然（清新、自然）
- 复古（经典、怀旧）

**用户操作**：
1. 系统根据产品风格自动匹配美学风格
2. 设计美学镜头（画面构图、色彩搭配）
3. 生成美学脚本

**年龄适配**：
- 婴幼儿可用（静态美学展示）
- 所有年龄段可用

**适用场景**：
- 美学产品（艺术品、家居装饰、高端服装）
- 提升品牌形象

---

### 2.9 Product Showcase 策略（产品展示）

**功能**：多角度展示产品细节，信息丰富

**展示类型**：
- 有模特展示（模特佩戴/使用）
- 无模特展示（平铺图、细节图）
- 局部出镜（产品细节特写）

**用户操作**：
1. 系统根据产品类别自动判断展示类型
2. 设计多角度展示镜头
3. 生成产品展示脚本

**适用场景**：
- 产品详情页视频
- 功能演示视频
- 细节展示视频

---

### 2.10 Story Theme 策略（主题叙事）

**功能**：故事线贯穿产品，完整剧情体验

**创作方法**：
```
热点事件 × 情感原型 → 生成故事主题 → 完整故事线
```

**用户操作**：
1. 系统选择热点事件（如"职场奋斗"）
2. 选择情感原型（如"励志原型"）
3. 热点 × 原型碰撞 → 生成故事主题（"职场励志"）
4. 设计完整故事线（产品自然植入）
5. 生成故事脚本

**年龄适配**：
- 婴幼儿/幼儿：不可用（需要角色主动参与叙事）
- 儿童及以上：可用

## 三、核心原理

### 3.1 智能搭配推荐原理

**联网趋势搜索**：

系统联网搜索最新时尚趋势，实时获取搭配灵感：
- 搜索当前季度流行穿搭趋势（如 2026 春季流行穿搭）
- 抓取时尚博主、明星街拍、社交媒体热门搭配
- 分析流行元素（色彩、款式、材质、风格）

基于最新趋势生成搭配方案：
- 当前流行色彩搭配（如 2026 春季流行色：柔和粉色、自然绿色）
- 流行款式组合（如 宽松上衣 + 修身下装）
- 热门风格推荐（如 都市极简、自然松弛）

**同款不同色自动识别**：

系统自动检测同款不同色的服装：
- 上传同一衬衫的黑色、白色版本
- 系统自动关联为同一变体组
- 生成搭配方案时推荐"换个颜色试试"

**AI 视觉识别增强**：

系统通过 AI 视觉识别服饰属性：
- 类别识别（上衣、下装、配饰）
- 款式识别（风格、版型、领型）
- 颜色识别（主色、配色）

结合联网趋势数据，智能匹配：
- 当前流行趋势 + 服饰属性 → 最佳搭配方案
- 趋势风格 + 用户服饰 → 推荐搭配组合

角色预设推荐：
- 根据搭配风格推荐角色方向
- 匹配当前流行趋势（如 2026 春季都市女性风格）
- 推荐姿势、推荐背景（契合趋势氛围）

---

### 3.2 角色五视图生成原理

**实时爆款模特脸特征分析**：

系统每天实时获取小红书、抖音等平台爆款模特脸特征：
- 抓取爆款模特照片（点赞量高的热门模特）
- AI 分析爆款特征（眼型、鼻型、肤色、发型等）
- 提取爆款元素（如近期流行"杏仁眼+小翘鼻+暖米肤色")

爆款特征实时注入：
- 热门模特脸特征 → 审美特征库
- 爆款元素组合 → 角色生成提示词
- 流行趋势特征 → 自动推荐给用户

**审美特征库系统**：

系统维护实时审美特征库，每日更新：
- 眼型特征（当前流行：宽眼型、杏仁眼）
- 眼色特征（当前流行：浅棕色、深棕色）
- 鼻型特征（当前流行：精致小鼻、鼻头微翘）
- 肤色特征（当前流行：暖米色、健康光泽）
- 发型特征（当前流行：柔软波浪、自然直发）

**爆款评分机制**：
- 爆款评分 ≥ 0.8：爆款特征（优先使用）
- 爆款评分 < 0.8：次要特征（备选使用）

**用户反馈动态调整**：
- 高分角色（≥4星）：提升特征爆款评分
- 低分角色（≤2星）：降低特征爆款评分
- 过时特征自动淘汰

**智能混血优化**：
- 特定地区自动混血（香港、新加坡 → 亚洲+欧洲混血）
- 单一种族30%概率轻微混血（优化视觉多样性）

**五视图生成流程**：
1. 用户输入：年龄、种族、风格
2. 特征库提取：主流审美特征
3. 随机组合：保证多样性
4. 生成五视图：正面/侧面/背面等多个视角
5. 用户评分：反馈调整特征库

---

### 3.3 年龄驱动策略选择原理

**年龄分组定义**：

| 分组 | 年龄范围 | 能力特征 |
|------|---------|---------|
| 婴幼儿 | 0-3 岁 | 无法站立、无主动动作能力 |
| 幼儿 | 4-6 岁 | 能站立行走，pose 能力有限 |
| 儿童 | 7-12 岁 | 能配合拍摄，可做简单展示 |
| 非少年 | 13-17 岁 | 大部分动作可用，青春风格 |
| 成人 | 18+ 岁 | 所有动作可用 |

**策略排除规则**：

系统根据年龄自动排除不合适的策略：
- **婴幼儿**：只能静态展示（Library, Aesthetic）
  - 排除 Fashion（需要 T 台走秀 pose）
  - 排除 Effectiveness（需要穿搭变身动作）
  - 排除 Emotion Archetype（需要主动参与剧情）
- **幼儿**：可做简单互动（Library, Aesthetic, Custom 亲子类）
  - 排除 Fashion（需要专业 pose）
  - 排除 Story Theme（复杂叙事不适合）
- **儿童及以上**：所有策略可用

---

### 3.4 热点系统原理

**每天自动获取热点信息**：

系统每天自动抓取各个平台热点数据：
- 抖音热点榜（视频热点、话题热点、音乐热点）
- 小红书热门笔记（穿搭热点、美妆热点、生活方式热点）
- 微博热搜榜（社会热点、娱乐热点、明星热点）
- 百度热搜榜（网络热点、新闻热点）
- B站热门视频（年轻文化热点、二次元热点）

**热点智能分析**：

每天分析热点数据，提取可创作元素：
- 热点趋势分析（哪些话题正在上升）
- 爆款内容拆解（热门视频的脚本结构、镜头节奏）
- 种草适配度评分（是否适合植入产品）
- 时效性判断（热点持续时间、最佳创作时机）

**热点特征库建立**：

每天更新热点特征库：
- 热门话题标签库（如 #职场穿搭、#春季美妆）
- 爆款脚本结构库（如"开场3秒吸引 → 痛点展示 → 产品植入"）
- 流行镜头节奏库（如"快节奏剪辑、慢镜头展示")

**脚本策略自动匹配**：

每天自动匹配热点与脚本策略：
- Realtime 策略：当天热点事件 → 时效性脚本
- Story Theme 策略：热点 × 情感原型 → 故事主题
- Video 策略：热门视频结构 → 借鉴脚本
- Custom 策略：小红书生活方式热点 → 场景化脚本
- Fashion 策略：小红书穿搭热点 → 时尚大片脚本

**热点推荐机制**：
- TopN 筛选：取各平台热榜前 N 个热点
- 适配评分筛选：只推荐高适配热点
- 时效性排序：优先推荐上升期热点
- 多平台组合：综合各平台热点，避免单一平台偏见
